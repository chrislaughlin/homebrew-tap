# Homebrew tap

Homebrew formulae maintained by Chris Laughlin.

```bash
brew tap chrislaughlin/tap
brew trust --formula chrislaughlin/tap/local-ai-advisor # Homebrew 6+
brew install local-ai-advisor
```

Or install directly:

```bash
brew install chrislaughlin/tap/local-ai-advisor
```

If Homebrew reports an untrusted tap, run
`brew trust --formula chrislaughlin/tap/local-ai-advisor`, then repeat the
install command.
